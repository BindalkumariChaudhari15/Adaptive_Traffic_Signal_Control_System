# Adaptive_Traffic_Signal_Control_System
This Project implements a pedestrian-centric adaptive traffic signal control simulation using queue based priority scheduling algorithm.

## Features
- Vehicle queue management
- Pedestrian request handling
- Adaptive signal control
- Performance evaluation
- Traffic simulation

## Technologies used
- Python
- NumPy
- Matplotlib

## Performance Evaluation
- Average pedestrian waiting time
- Average vehicle delay
- Throughput
- Queue length
- Fairness index

Run main.py.
